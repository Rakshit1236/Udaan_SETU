## Need to run or develop locally?

1. Clone this repository.
2. Ensure you have **Node.js** installed.
3. Install dependencies:
    ```
    npm install
    ```
4. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key.
5. Start the app:
    ```
    npm run dev
    ```
